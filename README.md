# Task 3 - Data Analysis Project

## 📌 Project Description
This project performs simple data analysis using Python and Pandas. It reads sales data from a CSV file and performs basic calculations like total sales, average sales, maximum and minimum values, and category-wise analysis.

## 🚀 Features
- Reads data from CSV file
- Calculates total sales
- Finds average, maximum, and minimum sales
- Performs category-wise analysis using grouping

## 🛠 Technologies Used
- Python
- Pandas
- CSV file handling

## ▶️ How to Run
1. Install pandas:
   pip install pandas

2. Run the program:
   python main.py

## 📊 Output Example
- Total Sales: 232000  
- Average Sales: 23200  
- Max Sales: 50000  
- Min Sales: 10000  
- Category-wise sales breakdown displayed

## 📁 Files Included
- main.py
- sales_data.csv
